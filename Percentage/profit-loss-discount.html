Watch the base! Mark-up % and profit % are both percentages of the cost price; discount % is a percentage of the marked price. Same number, different base → different dollars.

Part 3Worked examples from class

Profit % — buy at $150, sell at $285
\text{Profit} = 285 - 150 = \$135
\text{Profit}\% = \frac{135}{150} \times 100\% = 90\%
Loss % — cost $1650, sold for $990
\text{Loss} = 1650 - 990 = \$660
\text{Loss}\% = \frac{660}{1650} \times 100\% = 40\%
Discount % — marked $680, sold for $476
\text{Discount} = 680 - 476 = \$204
\text{Discount}\% = \frac{204}{680} \times 100\% = 30\%
Full journey — cost $1200, mark-up 60%, then sold at 88% of marked price
\text{Marked} = 1200 \times (1 + 60\%) = 1200 \times 1.6 = \$1920
\text{Selling} = 1920 \times 88\% = 1920 \times 0.88 = \$1689.6
\text{Profit} = 1689.6 - 1200 = \$489.6 \,,\quad \text{Profit}\% = \frac{489.6}{1200} \times 100\% = 40.8\%
