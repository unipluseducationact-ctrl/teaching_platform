Part 3"y% of x" — take y of the 100 parts

I am y% of x → x \times y\%.
Cut x into 100 equal parts; keep y of them. Each tiny slice in the bar below is 1% of x.
25%
0x = 320

Example from class: 32.5\% \text{ of } 320 = 320 \times \tfrac{32.5}{100} = 104, and 5\% \text{ of } 30 = 30 \times 0.05 = 1.5.

Part 4"y% more than x" — the base is x

I am y% more than x → the extra part is x \times y\%, because it is y% of x — x is the base (100%).
Original + Increase = New: x + x \times y\% = x \times (1 + y\%)

Don't mix them up: the increase is x \times y\% (a part), while the new value is x \times (1 + y\%) (the whole thing). "12% more than 1200" → increase 1200 \times 12\% = 144, new value 1200 \times 1.12 = 1344.
